# Bank-Managment-System-C
Bank Management System in C using structures and file handling to manage bank accounts. Features include creating accounts, deposit, withdrawal, balance enquiry, displaying all accounts, and deleting accounts. Data is persistently stored for real-world simulation.
## Features
- Create account with unique account number
- Deposit and withdraw money with input validation
- Check account balance
- Display all accounts
- Delete an account permanently
- Menu-driven and user-friendly
- Persistent storage using file handling

## How to Run
1. Clone the repository or copy the code into `bank.c`
2. Compile the program:
```bash
gcc bank.c -o bank
